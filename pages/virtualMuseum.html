Categorias

Placa mãe
Categoria reservada a placa mãe, a peça que une os componentes internos de um computador.

Armazenamento
Componente responsável pelo armazenamento persistente de dados de um computador.
Componente Interno
Categoria reservada a qualquer componente interno de um computador, como por exemplo processador, memória principal, etc.
Periférico

Impressora
Categoria reservada para dispositivos de impressão.
Telemóvel

Computador

Notebook
Categoria reservada a computadores portáteis.

Placa mãe

De modo resumido, as placas-mãe são componentes importantes que fornecem conexões entre os componentes internos do computador. Com a evolução da tecnologia, algumas dessas conexões foram aprimoradas, possibilitando um aumento na eficiência dos componentes conectados nela.

Antigamente, inclusive, a placa mãe tinha mais responsabilidades. Era possível encontrar, por exemplo, placas-mãe com um circuito responsável pelo processamento gráfico
