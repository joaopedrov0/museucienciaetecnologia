# Museu ciência e tecnologia

Olá, futuro bolsista ou curioso

Esse repositório é destinado ao site do museu ciência e tecnologia, um projeto de extensão da professora Flávia Beatriz, do IFSP - Campus Jacareí

O site foi feito em React, e caso você vá fazer manutenção deve se atentar a alguns pontos

1. Envio de emails: feito com EmailJS.
2. Dados dos itens presentes no museu em `src/data/hardware.ts` (Imagem em `/img`)
3. Dados dos dias que o museu será exposto `src/data/schedule.ts`.
4. Todas as rotas disponíveis estão em `/src/App.jsx`, verifique todas as funcionalidades.
